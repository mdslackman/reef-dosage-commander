# Aquarium Dosage Commander v7.5
A Python tool for reef aquarium water chemistry management.

### Key Features:
- **pH Guard:** Analyzes current pH before Alkalinity dosing to prevent dangerous spikes.
- **Precipitation Safety:** Checks Magnesium and Calcium foundation before allowing high Alkalinity changes.
- **Auto-Logging:** Saves every test and dose to `aquarium_data.csv` for trend analysis.
- **Speed Limits:** Enforces daily maximum rise limits to protect sensitive livestock.

### How to Run:
1. Ensure Python 3.x is installed.
2. Run `python reef_commander.py`.
3. Follow the on-screen prompts to log data or calculate doses.
